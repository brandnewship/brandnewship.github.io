Cuyahoga Valley Career Center Coding School

There's a new way to become a developer. Are you interested in becoming one?

What is it?

Through a 14-week crash course, you'll learning the ins and outs of web application development, including programming basics, frontend development, server and API development, and many other phrases and terms you'll know after the program.

What Will You Learn?

JavaScript

Using JavaScript, we'll teach you programming basics, data manipulation, and browser scripting. Unlike other programs, we'll teach you how to master this one language as your primary language, and show you how you can use this in a multitude of industries.

Databases

We'll teach you the basics of database management and administration. We'll show you how relational databases work using one of the most popular open-sourced database engine, MySQL, so you can apply that knowledge to any product or company.

Full Stack Development

Tying JavaScript, MySQL, and a handful of other tools and languages, you'll learn how to create user interfaces, animate elements, develop algorithms, read APIs, query data, build full-fledge applications, and be ready to push code as part of any development team.
